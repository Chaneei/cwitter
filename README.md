## 🔥 Introducing Project
#### 🗨 트위터 클론해보기!
🔓 firebase를 적극적으로 활용하여 DB, Auth 시스템 활용해보기

#### 🔎 나만의 글을 써보자
최초 화면에서 계정생성 후 로그인하여 트윗을 작성하고 실시간으로 작성되는 트윗을 만나볼 수 있습니다.
<br/>
글 작성 후 내가 작성한 트윗을 수정하고 삭제할 수 있게 설정하였습니다.

## 📁 폴더 구조
- src : FrontEnd로 화면 구성을 위해 routes, 공통 components, fbase.js를 통해 firebase설정을 해주었습니다.
- public : index.html SEO를 위해 meta태그 설정을 해주었고, favicon.ico로 파비콘 설정을 해주었습니다.

## 💎 How To Use?

#### start

```
npm install
npm start

```

#### build

```
npm run build
```

#### deploy
github를 사용했습니다
```
git remote add origin "주소"
git add .
git push origin master
```
